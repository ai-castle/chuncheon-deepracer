🚀 참가자 접수

8/1 (목), 10:00 접수 시작 (선착순)

참가자 접수 사전 준비사항 안내

대회 안내

본 대회는 춘천시에서 주최하는 AWS DeepRacer를 활용한 인공지능 강화학습 기반의 자율주행 자동차 AI 융합경진대회입니다.

  • 참가 대상
      - 인공지능 / 자율주행에 관심있는 전국 대학생 및 성인
      - 팀별 신청 (2~3명)
  • 참가 신청 (선착순)
      - 접수 기간 : 2024년 8월 1일 (목), 10:00 ~ 8월 7일(수), 10:00 까지
      - 접수 방법 : 온라인 접수 (bit.ly/ai-chuncheon)
      - 모집 규모 : 60개 팀 선착순
      * 추최측의 사정에 따라 60개 팀 이상 추가 선발 가능
  • 경기 방식
      - 예선 : 60개팀 / 온라인 대회
      - 본선 : 20개팀 / 온라인 대회
      - 결승 : 6개팀 / 오프라인 대회 (춘천시 커먼즈필드)
  • 지원
      - DeepRacer 계정 사용료 : 예선 (10시간), 본선 (30시간), 결승 (30시간)
      - DeepRacer 교육 : 4시간 x 2회
      - 딥레이서 차량 대여 (결승)
  • 시상 및 상품
    - 대상 (1팀) : 250만원 / 춘천시장상 / AWS 리인벤트 참관지원 예정
      * AWS 리인벤트 참관지원(2~3명)은 내부사정 및 현지상황에 따라 변경 또는 취소될 수 있음
    - 최우수상 (1팀) : 200만원 / 춘천시장상
    - 우수상 (2팀) : 100만원 / 강원정보문화산업진흥원장상
    - 장려상 (2팀) : 50만원 / 강원정보문화산업진흥원장상
  • 주최 및 주관
      - 주최 : 춘천시
      - 주관 : 강원정보문화산업진흥원
      - 후원 : AWS (Amazon Web Services)
